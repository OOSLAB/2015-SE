Main Description

작성항목

  1. 개요
    • 통합테스트 목적 : 테스트를 수행하는 목적을 간략히 기술한다.
    • 통합테스트 수행조직도 : 테스트 수행조직도를 작성한다.
    • 역할별 업무내용 : 통합테스트수행에 필요한 조직 및 구성원, 역할에 대하여 기술한다.
    • 테스트 일정 : 각 업무별 상세 일정에 대하여 기술한다.
    • 테스트 구성도 : 통합테스트가 수행되는 환경 구성을 가능하면 이미지로 표현한다
    • H/W, S/W 구성도 : 통합테스트가 수행되는 각 구성요소들의 H/W, S/W 사양에 대하여 상세히 기술한다. 테스트 환경은 가급적 실 환경과 동일하게 가져가고 부득이한 경우 실 환경의 축소판 형태로 테스트 환경을 구축하고 실 환경과의 비교를 통해 결과값을 환산한다
    • 테스트 지원 툴 : 통합테스트에 사용되는 Tool 및 사양에 대하여 기술한다.
    • 통합테스트시나리오 담당자 : 업무기능별 테스트 시나리오 및 통합테스트 수행 담당자를 기술한다.
  2. 통합테스트 결과
    • 종합 진척 및 결함 조치 결과 : 테스트 시나리오에 따른 테스트 진척 및 결함 조치결과를  기술한다.
    • 시나리오별 결함심각도 결과 : 시나리오별로 결함심각도를 분류기준에 따라 작성한다.
    • 심각도별 결함발생 및 조치결과 : 심각도별 결함발생, 조치, 미조치 건수 및 비율을 작성한다.
  3. 종합의견
    • 테스트 결과 분석 내용에 대한 종합적인 의견이나 개선방향을 기술한다.
