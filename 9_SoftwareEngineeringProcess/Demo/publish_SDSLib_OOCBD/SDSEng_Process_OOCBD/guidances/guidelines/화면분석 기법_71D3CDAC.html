Guideline: 화면분석 기법
Main Description

1.        개요 (Overview)

본 문서에서는 유스케이스를 기반으로 화면을 도출하는 방법과 각 화면의 레이아웃 구성 및 화면의 이벤트 정의에 관한 기법을 제공한다. 화면 분석 기법의 목적은 화면 분석 결과물이 화면 설계에 필요한 정보를 충분히 전달할 수 있도록 하는 데 있다.

 

2.        기법 적용시 유의사항 (Consideration)

프로젝트의 특성상 화면 분석 표준과 본 기법의 내용이 서로 맞지 않는 경우, 프로젝트의 화면 분석 표준을 우선해야 한다. 또한, 화면의 특성상 고객이 직접 시스템과 대면하는 접점이고 시스템에 대한 만족도를 크게 좌우하는 부분이므로 사용 편의성 면에서 고객 의견이 적극적으로 반영될 수 있어야 한다.
