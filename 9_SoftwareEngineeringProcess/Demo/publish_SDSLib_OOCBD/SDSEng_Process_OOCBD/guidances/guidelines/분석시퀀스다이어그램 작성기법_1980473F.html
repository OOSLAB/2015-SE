Main Description

1. 분석시퀀스다이어그램 작성 절차


 

2. 입력물

유스케이스 정의서 ::= 시나리오

시나리오 ::= 문장

문장 ::= 명사 + 동사 + 부사


3. 출력물

분석시퀀스다이어그램 ::= 객체 + 메시지 

메시지 ::= 객체 + 오퍼레이션 + 파라메터


4. 매핑 가이드

4.1. 유스케이스 정의서 시나리오의 명사를 중심으로 객체를 추출한다. 

4.2. 해당 객체와 연관된 동사를 중심으로 메시지를 추출한다.

     - 메시지 포맷은 대상객체, 오퍼레이션, 파라메터로 구성된다. 

     - 메시지 파라메터는 동사의 수식어로부터 추출한다.
