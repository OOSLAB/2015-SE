Description
Main Description

작성항목

  1. 기관명 : 프로젝트를 의뢰한 기관에 대해서 기재한다.
  2. 운영일자 : 시범운영이 이루어지는 일자를 기재한다.
  3. 지원기간 : 시범운영을 지원한 기간을 기재한다.
  4. 지원자 : 지원자명 및 지원자의 서명을 기재한다.
  5. 근무일지 : 근무 내역(근무시간, 수행업무, 비고 등)에 대해서 기술한다.
  6. 처리실적 : 시범운영시 시스템의 처리실적(처리업무명, 처리건수, 비고 등)에 대하여 기술한다.
  7. 문제점 및 개선 요구사항 : 문제점에 대한 개선사항(내역, 화면명, 요청자, 비고 등)에 대해서 기술한다.
  8. 지원확인 : 지원 내역에 대한 확인 사항(확인일자, 확인자 등)을 기재한다.
  9. 개선사항에 대한 의견 : 개산사항에 대한 의견 및 확인일자 및 확인자를 기술한다.
