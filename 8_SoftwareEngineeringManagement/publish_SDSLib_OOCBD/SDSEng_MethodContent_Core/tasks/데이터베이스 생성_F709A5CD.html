Task: 데이터베이스 생성
Disciplines: 컨버전및릴리이즈모델
Purpose
신 시스템의 데이터베이스를 실제 운영환경에 구축한다.

    • 데이터베이스
      Process Usage
      • 컨버전 및 릴리이즈  >  데이터베이스 생성
      Main Description

      수행 가이드(Task Guideline)

      1. 이행계획서 및 데이터베이스 설계 산출물을 참조하여 데이터베이스를 구축한다.
      2. 구축 후에는 고객에게 확인을 받도록 한다.
      Steps
      고객이 운영환경에서 사용할 DBMS를 설치한다.
      DBMS에 테이블을 생성하기 위해 테이블목록을 확인한 후 해당 테이블을 생성한다.
      생성된 테이블의 성능과 관련된 내용을 생성한다.(인덱스, 클러스터링 등)
      생성된 테이블의 완전성을 확인한다.
