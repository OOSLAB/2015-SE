Guideline: 단위테스트작성 기법
Main Description

1.        개요 (Overview)

본 기법에서는 프로젝트 개발 단계에서 단위 테스트 프레임워크를 이용하여 단위 테스트 코드를 작성하고 테스트 하는데 있어서 수행해야 할 절차 및 가이드 라인에 대해 설명한다.

 

2.        기법 적용시 유의사항 (Consideration)

본 기법은 JUnit 등 단위 테스트 프레임워크를 이용하여 클래스의 메소드 단위로 테스트 케이스를 정의하고, 테스트 코드를 작성하여 단위 테스트를 수행하는 유형인 경우 적용한다.

 

3.        관련 툴 (Related Tool)

JUnit

DBUnit

StrutsTestCase

MockEJB
